# ISM Sustainability Website

This project is a multi-page website designed to showcase sustainability initiatives at International School Manila (ISM), featuring an AI-powered chatbot.

## Getting Started

Follow these steps to run the website on your local machine.

### Prerequisites

- **Python 3.x**: Make sure you have Python installed. You can download it from [python.org](https://www.python.org/).

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/basnettk/ISM-Sustainability.git
   cd ISM-Sustainability
   ```

2. **Install dependencies**:
   Run the following command in your terminal to install the required Python libraries:
   ```bash
   pip install flask requests
   ```

### Running the Application

1. **Start the server**:
   Run the `app.py` script:
   ```bash
   python app.py
   ```

2. **Access the website**:
   Open your web browser and go to:
   [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Features

- **AI Chatbot**: Interact with ISM's sustainability chatbot powered by NVIDIA's AI API.
- **Sustainability Topics**: Explore pages on BYOC, Electricity, and Composting at ISM.
