# ISM Sustainability

Quick guide to running the website and the AI chatbot.

## Running the Website

1. **Install libraries**:
   ```bash
   pip install flask requests
   ```

2. **Start the app**:
   ```bash
   python app.py
   ```

3. **Open the site**:
   Go to [http://127.0.0.1:5000](http://127.0.0.1:5000)
